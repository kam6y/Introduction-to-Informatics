# Introduction-to-Informatics

Reveal.js slide scaffold for a 180-minute practical lecture.

## Quick start
- Open `index.html` in a browser (CDN required).
- Or run a local server:

```bash
python3 -m http.server
```

Then open `http://localhost:8000/`.

## Customize
- Edit `theme/custom.css` to tweak colors, typography, and layout classes.
- Duplicate sections in `index.html` to build your deck.
- Put images in `assets/` and reference them from slides.
